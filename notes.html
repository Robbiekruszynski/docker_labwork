CONTAINER

An isolated environment from running an application

VIRTUAL MACHINE

An abstraction of a machine (physical hardware)

Example: A mac can run a VM of windows and Linux on the same machine

HYPERVISOR

Software used to create and manage virtual machines

  • VirtualBox
  • VMware
  • Hyper-v (Windows only)
